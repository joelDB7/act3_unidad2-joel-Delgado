# promt 
"Genera el código completo de una aplicación Flutter en un solo archivo main.dart. La app debe usar rutas nombradas para las secciones: '/' (Inicio), '/pastillas', '/usuarios' y '/alarmas'. Incluye un Drawer con un UserAccountsDrawerHeader en color azul cielo fuerte (Color 0xFF0091EA), con una imagen de avatar circular desde la URL https://raw.githubusercontent.com/joelDB7/mis-imagenes/refs/heads/main/relojdeare.jfif. El nombre de la empresa 'A Tiempo' debe aparecer obligatoriamente en color negro, negrita y tamaño 18, mientras que el correo debe mostrar dirección, teléfono y email ficticios en tamaño 12. Los elementos del menú (ListTile) deben usar los iconos home, medication, people y hourglass_empty (para Alarmas) en el mismo tono azul. Todas las páginas deben tener un AppBar azul cielo fuerte con el título de la sección y el icono de menú en blanco. En el centro de cada pantalla, muestra una imagen de 200x200 con bordes redondeados (ClipRRect) obtenida de las URLs de GitHub correspondientes (relojdeare.jfif para Inicio, ketorolaco.jfif para Pastillas, usu.png para Usuarios y tilin.jfif para Alarmas) y debajo un texto que diga 'Sección: [Nombre]' en azul fuerte y negrita, y no  le quites el "contacto" en la barra de navegar de arriba para abajo,direccion,telefono,contacto
